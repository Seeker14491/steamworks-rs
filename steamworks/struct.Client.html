Struct steamworks::Client

pub struct Client(_);
Expand description

The core type of this crate, representing an initialized Steamworks API.

It’s a handle that can be cheaply cloned.

pub fn init(steam_app_id: Option<u32>) -> Result<Self, InitError>

Initializes the Steamworks API, yielding a Client.

A Steam App ID can be provided, which functions as an alternative to using a steam_appid.txt file.

Returns an error if there is already an initialized Client, or if SteamAPI_Init() fails for some other reason.
